# rvl-cli

> A cli for quickly creating Reveal.js slideshows using MarkDown.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
npm i rvl-cli -g
```

## Usage

`demo.md`:

```markdown
---
title: My First Slideshow
theme: black
---

Yay

---

## Second Slide

Nice!

```

```sh
rvl demo.md
```
